This project is a design of a smart contract with allows the user to conduct a lottery.
The player can enter the contact by making a contribution og 0.2 ether(can change the value as per convenience).
The host of the lottery can only pick the winner.
The winner are picked using the pseudo ramdom number generator.
The list of players can also be viewed by any participant.
All the pooled money will be transfered to the winner of the lottery.
After the winner is picked the whole contract is restarted.
